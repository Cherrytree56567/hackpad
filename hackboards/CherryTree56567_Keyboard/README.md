# MX Keyboard
A cool Keyboard made in KiCad, OnShape and Blender

## Info
 - 0.25mm Tolerance
 - Black Case

## Features
 - USB C Hub
 - RP2040
 - 61 Keys
 - Cherry MX Switches
 - Mill Max Hot Swappable Switches
